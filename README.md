## Overview
This project is an interactive Power BI dashboard designed to analyze sales, profit, orders, and customer data across regions, categories, and time periods.

## Features
- KPI metrics: Total Sales, Profit, Orders, Returns, Growth
- Sales analysis by Month, Quarter, Region, and Segment
- Top products and sub-category performance
- Customer insights (Top customers, customer count by region)
- Return and profit analysis
- Year-wise comparison

## Filters
- Region: East, West, North, South
- Category and SubCategory
- Segment: Consumer, Corporate, Home Office
- Year: 2024, 2025

## Pages
- Main Report: Overall summary
- Region Sales Report: Region-wise analysis
- Sales Details: Product and category insights
- Customer Details: Customer analysis

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
